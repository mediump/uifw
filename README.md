C++ UI framework based on SDL3

----

Features:
 - ECS-based domain graph
 - MSDF font rendering
 - Dependency-free text shaping/layout
 - Flex(-ish) layout containers
 - Support for basic styling (outlines, corner rounding)
 - Instanced rendering for sprites and glyphs
 - Support for Vulkan, D3D12, and Metal
